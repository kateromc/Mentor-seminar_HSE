
Создание временного alias в скрипте
Скрипт zadanie7.sh содержит команду:
alias ll='ls -la'

После запуска скрипта через source zadanie7.sh вывод команды ll показывает содержимое текущей директории в длинном формате:
total 36
drwxr-xr-x 1 huawei 197121    0 Oct 31 21:54 ./
drwxr-xr-x 1 huawei 197121    0 Oct 30 22:18 ../
-rw-r--r-- 1 huawei 197121   75 Oct 20 21:50 DZ1.md
...

Вывод: временный alias работает сразу после запуска скрипта.

Создание постоянного alias:
Чтобы сделать alias постоянным, в скрипте добавлена команда:
echo "alias ll='ls -la'" >> ~/.bashrc
source ~/.bashrc

Объяснение:
Файл ~/.bashrc выполняется при каждом запуске терминала Bash.
Добавление команды alias в этот файл делает её постоянной.
Команда source ~/.bashrc применяет изменения сразу в текущей сессии.

Проверка работы alias
После применения изменений через source ~/.bashrc можно использовать ll как обычную команду:
ll
Она выводит содержимое текущей директории в длинном формате.

Демонстрация автодополнения
Пример автодополнения на команде cd:
Ввод: cd Ment
Нажатие Tab → Bash подставляет: cd Mentor-seminar_HSE/
Нажатие Enter → переход в директорию.
Вывод: автодополнение работает и ускоряет работу с длинными именами директорий.
